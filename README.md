# PromptCraft – System Prompt Block Composer

> Free, browser-local tool for assembling structured system prompts from modular blocks.
> No signup. No upload. Zero storage. Single HTML file.

🔗 **[Open Tool →](https://AyushmanDhar.github.io/promptcraft/)**

---

## What it does

Drag-and-drop block editor for building system prompts used with ChatGPT, Claude, Gemini, and other LLMs.

Blocks: Role · Context · Task · Constraints · Output Format · Persona · Examples · Tools · Memory · Custom

Output formats: Plain text · XML tags · Markdown

Saves 5 named slots to localStorage. Copy to clipboard or export `.txt`.

---

## Why

Every serious prompt engineering tool requires an account, a monthly subscription, or a backend.
This is a single `.html` file. Open it, build your prompt, close it. Nothing leaves your device.

---

## Use cases

- AI developers building system prompts for agents or assistants
- Indie hackers prototyping LLM-powered apps
- Prompt engineers managing multiple persona/instruction sets
- Anyone tired of writing system prompts in Notepad
